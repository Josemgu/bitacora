FASE 0 Oct 2026

Base del homelab

Proxmox + pfSense/OPNsense como VM desde el día 1. Las 4 VLANs existen aunque estén vacías.

  • Hypervisor y topología base
  • pfSense/OPNsense perimetral
  • VLANs: Management, Servers, Users, DMZ
  • Repo GitHub + primer post en LinkedIn
homelab-cloud-security
FASE 1 Nov 2026 – Mar 2027

Linux + Networking real

Hardening real de Ubuntu y las 4 VLANs implementadas de verdad en pfSense — reglas aplicadas, no un diagrama.

  • CIS Benchmark + SSH solo por llave
  • rsyslog centralizado
  • Backups probados con restore real
  • Reglas de firewall inter-VLAN
  • Wireshark: tráfico entre VLANs
linux-hardened-server homelab-network-segmentation
FASE 2 Abr 2027 – Ago 2027

Windows Server + Ofensiva básica

AD ficticio "Ortiz Technologies" con 100 usuarios, y un ataque simulado contra tu propio dominio.

  • Dominio con OUs por departamento
  • GPOs: LAPS, SMBv1 off, PowerShell restringido
  • ADCS interna + servidor de archivos con DFS
  • Kerberoasting / AS-REP Roasting documentado
ad-lab-attack-detection CompTIA Security+
FASE 3 Sep 2027 – Feb 2028

AWS + Infraestructura como código

Multi-cuenta con Organizations, IAM de mínimo privilegio, Terraform con state remoto y pipeline real.

  • AWS Organizations: cuentas dev/prod
  • VPC, subnets, NAT, Security Groups, NACLs
  • State remoto: S3 + DynamoDB locking
  • GitHub Actions: plan en PR, apply en merge
aws-infra-as-code AWS Solutions Architect Associate
FASE 4 Mar 2028 – Ago 2028

Docker + Kubernetes con carga real

Cluster de 1 master + 2 workers sobre Proxmox, desplegando el stack que usará la Fase 5.

  • Cluster K8s real en el homelab
  • GitOps con ArgoCD
  • Trivy escaneando imágenes en el pipeline
k8s-homelab-platform
FASE 5 Sep 2028 – Feb 2029

SOC casero + AWS Security avanzado

Wazuh + Suricata + Grafana ingiriendo todos los logs. Ejercicio purple team contra tu propio homelab.

  • SOC completo, no herramientas sueltas
  • Cadena de ataque completa + detección
  • GuardDuty, Security Hub, Config, alertas SNS
  • Reporte de incidente publicado
homelab-soc incident-report
FASE 6 Mar 2029 – Jun 2029

IA aplicada a seguridad

RAG local que ingiere las alertas reales de tu propio Wazuh — no un dataset genérico de ejemplo.

  • AI SOC Assistant: Ollama + vector DB
  • AI Cloud Assistant: revisión de Terraform
  • AI Homelab Assistant
ai-soc-assistant
FASE 7 Jul 2029 – Oct 2029

Preparación de búsqueda de empleo

No se construye portafolio desde cero — se pule lo documentado desde la Fase 0.

  • 8–12 repos ya documentados
  • CV en formato canadiense
  • Simulacros: explicar arquitectura, no teoría
